# h5+php7.4+MySQL聊天室

## 介绍

基于H5前端、PHP7.4+服务端与MySQL数据库搭建的轻量实时在线聊天室系统，原生无第三方IM框架，纯原生PHP+HTML+JS实现轮询实时聊天，自带完整后台管理、用户体系、IP封禁、头像上传、自定义主题、Emoji表情、图片发送等全套功能，支持私有化部署，适配虚拟主机、宝塔面板等PHP环境。

### 整体架构

1. **前端层**：原生HTML5 + CSS3 + jQuery，无Vue/React等重型框架，轻量化加载，适配PC/移动端，支持深色模式、自定义气泡配色、自定义聊天背景、消息音效、桌面通知、代码块渲染、图片预览、全套Emoji表情面板。

2. **服务端层**：PHP 7.4及以上，基于PDO操作MySQL，纯原生开发，无任何PHP框架，文件式单页面开发，接口分离处理消息拉取、发送、在线用户、用户资料、管理操作；使用Session完成登录鉴权，CSRF防护后台操作。

3. **数据库层**：MySQL 5.7+/8.0，字符集utf8mb4（完整支持emoji存储），包含用户、聊天消息、在线用户、IP黑名单、用户定位五张数据表，外键关联保证数据完整性。

4. **文件存储层**：本地文件存储，头像、聊天图片上传至 `uploads` 目录，系统日志存放 `logs`，静态资源存放 `assets`。

## 安装教程

1. 将项目全部源码上传至网站根目录，设置网站运行目录为源码根目录，给文件夹赋予读写权限。
2 浏览器访问域名 http://你的域名/install.php 进入一键安装页面。
3 填写数据库信息：数据库主机、数据库账号、数据库密码、自定义数据库名称（不存在会自动创建）。
4 设置后台管理员用户名、登录密码、展示昵称。
5 点击【开始安装】，程序自动完成：创建数据库、生成5张数据表、生成 config.php 配置文件、自动创建uploads/logs资源文件夹、写入管理员账号、生成安装锁 install.lock 。
6 安装完成后点击跳转登录页，使用管理员账号登录聊天室或后台。
 
重装说明：如需重新安装，手动删除根目录 config.php 与 install.lock 两个文件，重新访问install.php即可。

## 使用说明

### 普通用户功能

1. **注册登录**：支持账号注册，IP黑名单用户访问注册/登录页直接返回封禁图片；账号禁用无法登录。
2. **聊天发送**：支持纯文字、多行文本、代码块、本地图片上传发送，内置完整分类Emoji表情选择器。
3. **界面设置**：深色/浅色模式切换、自定义收发气泡颜色、自定义图片/纯色聊天背景、消息提示音效开关、音量调节、新消息桌面通知、自动滚动消息开关。
4. **个人主页**：修改头像、昵称、个性签名，支持修改登录密码。
5. **在线列表**：实时查看当前在线用户、展示用户昵称/头像/所在城市，区分管理员标识。
6. **消息权限**：普通用户仅可删除自己发送的消息。

### 管理员专属功能

1. **聊天权限**：可删除任意用户发送的全部消息。
2. **管理后台入口**：聊天室侧边栏进入管理面板。
3. **仪表盘**：统计总用户、活跃/封禁用户、IP黑名单数量、今日消息、今日活跃用户等数据。
4. **用户管理**：查看全部注册用户、禁用/解封账号、永久删除用户、一键封禁用户IP、自定义用户展示地理位置。
5. **消息溯源**：单独查看任意用户所有历史聊天记录，支持逐条删除。
6. **IP黑名单**：手动添加恶意IP封禁、解封已封禁IP，封禁IP无法访问登录/注册页面。

## 运行注意事项

1. 项目依靠前端3秒轮询拉取最新消息，15秒刷新在线用户列表，无需WebSocket，普通虚拟主机即可稳定运行。
2. 头像、图片上传单文件限制10MB，支持jpg/png/gif/webp格式。
3. 自动通过ip-api接口获取用户IP对应城市，无外网环境会默认显示"未知"。
4. 数据库密码、账号明文保存在config.php，线上环境禁止对外泄露该文件。
